  • ✔
    Accesibilidad (WCAG) es un Derecho En el ámbito público, una web sin accesibilidad es como un edificio sin rampa.
  • ✔
    Diseño Responsivo Real No diseñamos para monitores, diseñamos para contextos (ej. bajo el sol, con baja señal).

🏥 Caso local: Salud Pública

Imagina el portal de turnos de un hospital regional. No es solo una página; es una infraestructura. Si la Arquitectura de Información falla, el paciente no llega a su cita. Un error de diseño aquí es un obstáculo para la salud.

⚠ Regla de oro (Heurísticas)

Tú no eres el usuario. Debemos abandonar nuestros sesgos y gustos personales. La UI es la "cara", pero la UX es el "cerebro" que previene la frustración. Falla rápido y barato usando herramientas como Figma antes de escribir una sola línea de código.
